Configurar Firebase

O Firebase é o banco de dados gratuito do Google que armazena as mensagens em tempo real. Configure uma vez só.
  1. Acesse console.firebase.google.com e crie um projeto
  2. Clique em Realtime Database → Criar banco → modo de teste
  3. Vá em Configurações ⚙ → Seus apps → ícone Web </>
  4. Copie o objeto firebaseConfig completo e cole abaixo
